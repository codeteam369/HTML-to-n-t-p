Các bộ chọn của Css

Css có rất nhiều bộ chọn cho nhà phát triển tùy chỉnh

Một số cách chọn element(phần tử) trong css cơ bản:

  • Chọn theo tên thẻ(tag); tag
  • Chọn tất cả; *tag
  • id-identify (định dạng duy nhất) theo quy chuẩn là không cho phép đặt trùng id; #id
  • class ở đây hiểu là cùng kiểu, khi muốn tác động vào nhiều phần tử nhất định ta dùng class; .class

Một số lưu ý về đặt biến: id và class trong html hoặc variable(biến) trong python thì không cho phép đặt số ở đầu
